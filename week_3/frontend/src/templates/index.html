THREE WEEKS · THREE LAYERS

What We Built

An end-to-end AI pipeline — from raw web scraping to a conversational interface — built incrementally over three weeks.

Week 1 — Data Pipeline

ETL · SQLite · Bronze / Silver / Gold

Scrape job listings from JobStreet Malaysia and push them through a medallion ETL pipeline. Raw HTML lands in Bronze, cleaned records in Silver, and analytics-ready rows in the Gold SQLite database. 98 tech jobs covering AI, data engineering, and software roles.

Python SQLite BeautifulSoup
Week 2 — LLM Integration

Ollama · Gemini · MCP Server

Connect local models (DeepSeek-R1, Llama 3.1, Phi-3, Qwen) via Ollama and cloud models (Gemini 2.5 Flash) via Google's API. A unified prompt_model() function routes requests to the right backend. An MCP server exposes job data tools to AI clients.

Ollama Gemini MCP pydantic
Week 3 — Web Application

FastAPI · Bootstrap · Docker Compose

A two-service web stack: a frontend that serves a Bootstrap chat UI and a backend that wraps the LLM and DB queries. PDF resumes are parsed in-browser with PDF.js and sent alongside messages. Chart.js renders live job market analytics.

FastAPI Bootstrap 5 Chart.js Docker

SETUP GUIDE

Quick Start

From zero to running in under 10 minutes

1
Clone the repository
git clone https://github.com/AsynchronousNotAvailable/kyouth-data-ai
cd kyouth-data-ai/week_3
2
Run the Week 1 data pipeline first time only
# Scrape jobs and build the SQLite database
cd ../week_1 && uv run python main.py
# Output: week_1/data/3_gold/jobs.db
3
Pull the LLM model into Ollama
# Start Ollama container and pull the model
cd week_3
docker compose up ollama -d
docker compose exec ollama ollama pull llama3.1:latest
4
(Optional) Add a Gemini API key for cloud models
# week_3/backend/.env
GEMINI_API_KEY=your_key_here
5
Start the full stack
docker compose up --build

# Services:
✓ Frontend → http://localhost:8000
✓ Backend → http://localhost:8001
✓ Ollama → http://localhost:11434

WHAT YOU GET

Features

AI Resume Chat

Ask the AI anything about your resume, skill gaps, or job market trends. Context-aware with a career-focused system prompt.

PDF Resume Upload

Upload your PDF resume — text is extracted client-side with PDF.js and attached to every AI message for personalised advice.

Jobs Dashboard

Interactive Chart.js visualisations — role distribution pie chart and job-by-location bar chart — powered by the live SQLite database.

Job Search

Full-text search across 98 Malaysian tech job listings by title, company, or tech stack. Results update as you type with a 300 ms debounce.

Local LLM First

Runs entirely offline using Ollama. Supports Llama 3.1, Phi-3, DeepSeek-R1, and Qwen 3.5 — no API key required.

Cloud LLM Option

Swap in Gemini 2.5 Flash with a single env var. The same prompt_model() interface routes by model name.

Fully Containerised

Three Docker services on a shared bridge network. One docker compose up command starts everything. No host network driver used.

Medallion Data Layer

Bronze → Silver → Gold ETL pipeline. Raw scrape data is cleaned, normalised, and promoted layer by layer into a queryable SQLite database.

WALKTHROUGH

How to Use It

Chat Interface
1

Open http://localhost:8000 — you land on the Chat tab with a greeting message.

2

Click the paperclip icon and upload your PDF resume. PDF.js extracts the text locally — nothing is uploaded to an external server.

3

Type your question and press Enter. Your message + extracted PDF text are sent as JSON to POST /chat on the backend.

4

The backend builds a career-focused prompt, calls Ollama (or Gemini), and streams the reply back to the chat bubble.

Jobs Dashboard
1

Click the Jobs Dashboard tab in the navbar. Charts load lazily on first open from GET /api/stats.

2

The pie chart shows the role category breakdown (AI Engineer, Data Engineer, Software Engineer, etc.).

3

The horizontal bar chart ranks Malaysian states by number of available jobs — Kuala Lumpur leads, followed by Selangor and Penang.

4

Use the search bar to filter all 98 job listings by title, company name, or tech stack in real time.
